Project Overview

This project involved developing multiple deep learning models to accurately predict pneumonia cases using chest X-ray images. The models were trained to identify patterns indicative of pneumonia, helping in early diagnosis and treatment.

Key Features and Functionality

  • Data Preprocessing: Applied image enhancement techniques and data augmentation to expand the dataset.
  • Model Development: Implemented multiple algorithms: Deep Belief Network (DBN), Deep Neural Network (DNN), Self-Organizing Maps (SOM), Support Vector Machine (SVM), and Feedforward Neural Network (FNN).
  • Performance Metrics: Evaluated using accuracy, precision, recall, F1-score, and ROC-AUC curves.

Technologies and Tools Used

Programming Languages: Python

Frameworks: TensorFlow, Keras, PyTorch

Tools: Jupyter Notebook, OpenCV

Libraries: NumPy, Pandas, Scikit-Learn, Matplotlib

Development Process

Step-by-step approach:

  1. Data Collection: Sourced chest X-ray images from public databases.
  2. Data Preparation: Cleaned and labeled data; split into training, validation, and test sets.
  3. Model Training: Built and fine-tuned DBN, DNN, SOM, SVM, and FNN models.
  4. Evaluation and Comparison: Compared model performance using key metrics.

Model Performance Comparison

The following table summarizes the accuracy achieved by each model:

  • Deep Belief Network (DBN): 91.18%
  • Deep Neural Network (DNN): 83%
  • Self-Organizing Maps (SOM): 77%
  • Support Vector Machine (SVM): 88%
  • Feedforward Neural Network (FNN): 81%

Challenges and Learnings

Faced challenges with data imbalance, overfitting, and computational efficiency, which were tackled using various techniques such as SMOTE, dropout layers, and GPU acceleration.

Project Outcomes and Impact

The DBN model achieved the highest accuracy of 91.18%, outperforming other models. These results suggest the DBN model's potential for reliable early pneumonia detection.

Next Steps or Future Improvements

Plans include integrating the model into real-time diagnostic tools and exploring other deep learning models for further improvement.
